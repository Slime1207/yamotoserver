# 📗 地圖繪 Image on Map

#### 如何創建一個地圖繪？

創建一個地圖繪需要一張空白地圖。然後輸入指令 : `/imageframe create <名稱> <URL> <寬度> <長度>`

輸入完畢后，檢查是否有跳出成功的字眼。

#### 大小建議

大小建議為 `(長度)128 * (寬度)*128`，因爲地圖本身大小爲 128 x 128 格。
